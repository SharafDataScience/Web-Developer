# 1- project 2: Swansea Design Studio – Website Project (Virual)
Swansea Design Studio is a conceptual web design company based in Swansea. This project was created to simulate a real-world business environment and demonstrate practical skills in modern web design and development.
The website focuses on delivering a professional digital presence for a creative agency, offering services such as Web Design, SEO, Graphic Design, and Mobile Application development.
A key feature of this project is the integration and customization of animation effects using the Ele Animation plugin, implemented independently to enhance user interaction and create a smooth, engaging browsing experience.<br>
